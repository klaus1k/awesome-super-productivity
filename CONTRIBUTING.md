# Contributing

Contributions are welcome! Here's how you can help.

## Adding a Resource

Please ensure your submission:

- Is related to Super Productivity
- Is not already in the list
- Has a working link
- Includes a brief description

## Guidelines

- Add new items at the bottom of the relevant section
- Use the format: `[Title](URL) - Brief description.`
- Check your spelling and grammar
- One pull request per resource (or group of related resources)

## Pull Request Process

1. Fork the repository
2. Add your resource(s)
3. Submit a pull request with a clear description

Thank you for contributing!
